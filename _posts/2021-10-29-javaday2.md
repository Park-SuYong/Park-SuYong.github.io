## 변수를 선언하고 원하는 문구를 출력하기

* 변수(variable)<br/>
  * 변수는 저장공간이다<br/>

* 변수의 선언<br/>
  * 자료형 변수명;<br/>

* 자료형 (저장공간의 종류)<br/>
  * 자료형	int	1, -2<br/>
  * 정수형	double	10.0, -9.3<br/>
  * 문자형	char	'a', '안'<br/>
  * 문자열	String	"", " ", "10", "안"<br/>

* 변수명 생성시 주의사항<br/>
  * 띄어쓰기 불가능 int my age //오류<br/>
    * int myAge //가능 - 가독성을 위해서 대문자로(카멜표기법)<br/>
    * int my_age //가능<br/>

* 특수문자 사용 불가능<br/>
  * _는 가능<br/>

  * 숫자로 시작하면 안된다<br/>
    * int 1num; //오류<br/>
    * int 10; //오류<br/>
    * int num1; // 가능<br/>
	
* 키워드 사용 불가<br/>
  * int int; //오류<br/>

* 권고사항 <br/>
  * 의미 있는 단어를 사용하자<br/>
  * 클레스는 대문자로 변수는 소문자로 시작하자 //구분하기 위해서<br/>

* 문자를 출력하기
  * print - 출력을 한다.
      int NUM; //선언
	  	double pi;
  		char score;
		
  		NUM = 10; //초기화
  		pi = 3.14;
  		score = 'a';
		
  		System.out.println(NUM);
  		System.out.println(pi);
  		System.out.println(score);
      
  * println - 출력을 하고 줄바꿈을 한다.
      System.out.print("안녕");
	  	System.out.print("방가");
  		System.out.println();
  		System.out.println("이러면 줄바꿈");

  * printf - 형식을 맞추어 출력할때 사용하는 메소드 - 형식지정자
      double pi = 3.14159;
		  System.out.printf("원주율은 %.2f입니다", pi);
